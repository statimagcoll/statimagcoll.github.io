These figures show "Target" and "Probability" images using p-value (p) or effect size (S) cluster thresholding for spatial extent inference associated with the paper Vandekar & Stephens, 2020. The effect size value is based on the index S (Vandekar, Tao, & Blume 2020). For p-value thresholding, the target effect size changes with the sample size. For effect size thresholding the target effect size is fixed and the p-value threshold becomes more stringent for larger sample sizes ("n"). The results are obtained using bootstrapping for 6 different sample sizes using 10,000 bootstraps from the ABIDE data set (Di Martino et al. 2014). The target images show regions where the expected value of the chi-squared statistic is above the cluster forming threshold for a given sample size, indicating that those regions are in the target set for the given sample size. The probability images show the probability that a given voxel is identified as being in the target set for a random sample.
